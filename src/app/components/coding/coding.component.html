Task Description

A binary number is represented as a series of 0s and 1s. In this challenge, the series will be in the form of a singly-linked list. Each node instance, a LinkedListNode, has a value, data, and a pointer to the next node, next. Given a reference to the head of a singly-linked list, convert the binary number represented to a decimal number.

Example

Linked list corresponding to the binary number (010011)₂ or (19)₁₀.

Output

fdfdf
